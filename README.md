# S&P 500 Market Explorer

Um dashboard interativo para explorar a composição e os fundamentos das 500 maiores empresas de capital aberto dos Estados Unidos.

## Sobre o projeto

Este projeto foi desenvolvido como parte do bootcamp de Data Analytics da TripleTen. O objetivo é aplicar habilidades de análise exploratória de dados e desenvolvimento de aplicações web com Python.

## Funcionalidades

- Filtro interativo por setor
- Distribuição de Market Cap (histograma)
- Market Cap vs Crescimento de Receita (scatter plot)
- Crescimento médio por setor
- Lucratividade por setor (EBITDA absoluto e múltiplo de valuation)
- Concentração do índice (Top 10 empresas)
- Desigualdade interna por setor (box plot)
- Outliers: empresas pequenas com alto crescimento
- Concentração geográfica por estado

## Tecnologias utilizadas

- Python
- Pandas
- Plotly Express
- Streamlit

## Como executar localmente

pip install -r requirements.txt
streamlit run app.py
