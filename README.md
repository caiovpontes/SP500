# SP500
Projeto de análise de dados utilizando informações de mais de 500 empresas do S&amp;P 500. O foco é explorar métricas financeiras, comparar setores e visualizar tendências de mercado através de gráficos interativos e filtros dinâmicos. Ideal para demonstrar habilidades em EDA, visualização de dados e manipulação de datasets reais.
